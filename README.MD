## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+ 
- MongoDB database
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd api-REST-Orange-Box
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   Create a `.env` file in the root directory:
   ```env
   PORT=3000
   MONGO_URI=mongodb://localhost:27017/orange-box
   # OR use MongoDB Atlas
   # MONGO_URI=mongodb+srv://username:password@cluster.mongodb.net/orange-box
   
   # Optional for deployment
   RAILWAY_URL=https://your-app.railway.app
   VERCEL_URL=https://your-app.vercel.app
   ```

4. **Run the application**
   ```bash
   # Development mode (with hot reload)
   npm run dev
   
   # Production mode
   npm run build
   npm start
   ```

5. **Access the API**
   - API Base URL: `http://localhost:3000/api/v1`
   - Documentation: `http://localhost:3000/api-docs`

## ğŸ›ï¸ Products API

### Create Product
**POST** `/products`

```json
{
  "name": "Premium Laptop",
  "price": 1299.99,
  "description": "High-performance laptop",
  "provider": "64a7b8c9d1234567890abcde",
  "stock": 50,
  "status": "active"
}
```

**Response (201)**
```json
{
  "success": true,
  "message": "Product created successfully",
  "data": {
    "_id": "64a7b8c9d1234567890abcde",
    "name": "Premium Laptop",
    "price": 1299.99,
    "description": "High-performance laptop",
    "provider": "64a7b8c9d1234567890abcde",
    "stock": 50,
    "status": "active",
    "createdAt": "2023-07-07T10:30:00.000Z",
    "updatedAt": "2023-07-07T10:30:00.000Z"
  }
}
```

### Get All Products
**GET** `/products`

**Query Parameters:**
- `page` (number): Page number (default: 1)
- `limit` (number): Items per page (default: 10, max: 50)
- `sortBy` (string): Field to sort by (default: createdAt)
- `order` (string): Sort order - asc/desc (default: desc)
- `search` (string): Search in name and description
- `status` (string): Filter by status (active/inactive/discontinued)
- `provider` (string): Filter by provider ID
- `minPrice` (number): Minimum price filter
- `maxPrice` (number): Maximum price filter
- `fields` (string): Comma-separated fields to return

**Example Request:**
```
GET /products?page=1&limit=10&status=active&minPrice=100&search=laptop
```

**Response (200)**
```json
{
  "success": true,
  "message": "Products retrieved successfully",
  "data": [...],
  "pagination": {
    "currentPage": 1,
    "totalPages": 5,
    "totalItems": 45,
    "itemsPerPage": 10
  }
}
```

### Get Product by ID
**GET** `/products/{id}`

**Response (200)**
```json
{
  "success": true,
  "message": "Product retrieved successfully",
  "data": {
    "_id": "64a7b8c9d1234567890abcde",
    "name": "Premium Laptop",
    "price": 1299.99,
    // ... other fields
  }
}
```

### Get Products by Provider
**GET** `/products/provider/{providerId}`

Same query parameters as "Get All Products" but filtered by provider.

### Update Product (Full)
**PUT** `/products/{id}`

Requires all fields in request body.

### Update Product (Partial)
**PATCH** `/products/{id}`

```json
{
  "price": 999.99,
  "stock": 25,
  "status": "inactive"
}
```

### Delete Product
**DELETE** `/products/{id}`

**Response (200)**
```json
{
  "success": true,
  "message": "Product deleted successfully"
}
```

### Delete Provider
